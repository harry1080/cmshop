==========
1.微信接口设置
==========
* 登录微信公众号平台 https://mp.weixin.qq.com  功能 -》高级功能 -》进入开发模式 =》输入微信通后台的Token值与URL地址  提交。
对接URL地址格式 ： 你的域名/wechat/weixintong-ent.php。如：http://www.abc.com/wechat/weixintong-ent.php， 提交成功后将 AppId、 AppSecret，信息填写到后台。

 注： Token:weixin (默认，可以修改但要与微信公众号里的值相同) 进入开发都模式成为开发者时，可以自定义填写Token值。
AppId、 AppSecret 是用来设置微信菜单的，如果没有说明你没有设置微信菜单的功能，后台则请留空。


==========
2.微信菜单设置
==========
* 公众号权限要求：认证订阅号、普通服务号、认证服务号。
* 首次接入微信通时，微信菜单不会自动生成，要在后台重新（生成）保存一次。(修改一次微信菜单24小时生效，详见微信官方说明)
* 微信菜单 最多包括3个一级菜单，每个一级菜单最多包含5个二级菜单。
* 一级菜单最多4个汉字，二级菜单最多7个汉字，多出来的部分将会以"..."代替。（菜单数量的限制是微信官方的限制）

菜单一，菜单二，菜单三，三个菜单选项，分别依次从左到右对应微信底部三个菜单。
一级菜单 值默认为空 不要更改。（因为在微信中，这个菜单如果有二级菜单，一级菜单则不需要点击）
* 类型：click文字  view网址（后面值填文字时用click,填网址时用view）
* 名称：在微信中显示的菜单名称
* 值：可以是中文、字母、网址（填写网址时，要注意使用view类型）

==========
3.微信通设置
==========
*手机站路径 mobile/ 默认
*手机站网址 http://www.a.com/  （注意要以 / 结尾）

如果绑定的是一个二级域名 如：URL是m.a.com  则
手机站路径：留空不填
手机站网址：http://m.a.com/

以下设置一般都默认就可以了。
*图片路径：local (表示图片为本地) server (远程图片)
开启远程后 通过微信菜单获取到的商品缩略图片直接用绝对地址 表示，文章图片不会变。

*搜索推荐：true (表示启用) false (表示关闭)
--》搜索推荐 搜索不到产品时随机推荐一个“精品”的产品，结果以文本的形式展示。有客服人工回复时建议关闭此功能。

*默认密码：每个微信粉丝关注会自动增加一个会员，密码就是指这个会员的初始密码。
--》会员名称就是微信昵称，这样微信用户也可以快速在PC上登录。

*重新绑定：true (表示启用) false (表示关闭)
--》重新绑定 是没有特别需要的时候建议关闭，默认是开启状态。开启会对用户体验不好。

*绑定会员模式：web (进入web绑定) step (原有的流程式绑定)

*微信OAuth:true (表示启用) false (表示关闭)
--》默认false。   必须是经过微信认证的服务号才能改成true，可以实现自动登录，详见下面说明与使用。

*显示下架商品：true (表示启用) false (表示关闭)

4.关注送红包（优惠券）
==========
******关注送红包是结合中移智能的红包功能
一、中移智能后台操作：促销管理-》红包类型-》添加-》线下红包-》发放
二、关注送红包-》选择要发送的红包
三、优惠券提示的内容在"语言设置"中修改。（优惠券全部信息是：优惠前言+优惠券号，到期时间+优惠券尾部）
四、（不能与关注图文回复 同时使用）例如 选择了 关注送多少红包。则用户关注会显示送红包的提示信息。如果不想要这个提示信息，则将选择项设置为空，点确定保存。
五、用户使用红包：则登录用户中心=》我的红包=》添加红包  输入 红包兑换码，验证成功后，就可以在购买商品时选择使用了。

6.语言设置（自定义语言设置）
==========
基本包含微信通中默认功能互动文字设置，可以修改内容但不能添加，以便于商家根据自己的行业设置更适合的文字提示。

其中 帮助说明 固定关键词help 加入到微信菜单中，用户点击获取后，可以看到目前可以输入哪些有用的关键词，或者能使用的功能。

==========
7.关键词自动回复
==========
关键词自动回复 分两种类型 纯文本回复规则 与 图文回复规则。
a.添加文本规则。
填写好规则名称，关键词多个则以空格来区分，消息类型 默认文字。内容中文字的消息可以使用超链接、商品或文章链接。

b.添加图文回复规则。
特别说明一下，图文回复 也可以设置两种 
b-1.关注自动图文回复（即用户关注微信公众号 自动回复一条图文消息） 
注：关注自动回复功能 一次只能设置一种回复类型。例如你设置了关注送红包信息，则关注回复文本信息与关注回复图文信息 则此时无效。默认是关注自动回复文本信息，如有设置关注送红包则优先关注自动回复红包信息，关闭关注送红包后，添加一条关注自动图文回复规则 则会用关注自动图文回复规则。

--在菜单"关键词自动回复"中添加图文新规则，关键词设置为：subscribe 。仅仅关注自动图文回复功能用这个关键词，其他不必。  (图片大小较好的效果为大图640*320，小图80*80)

b-2. 普通图文回复规则  添加方法同上，只是关键词不要填subscribe了，可以任意自定义关键词。文章可以多篇，也可以添加多条图文回复规则。

问题：文章里显示在微信上的缩略图片 是在中移智能后台文章管理=》编辑文章=》上传文件 这里上传缩略图

15.微信OAuth
==========
微信OAuth：是网页授权获取用户基本信息，微信通中主要用于安全的免登录，生成的调用网址，可以添加到菜单或者是自动回复中，微信粉丝点击可以直接登录。
首先在微信通设置菜单中 将微信OAuth项 修改成true。然后将现有默认的一条跳转网址规则中 修改为自己域名/mobile。复制 域名/wechat/oauth/wxch_oauth.php?oid=1  填写到微信菜单 跳转到网站首页的 值当中。

再到微信公众平台 服务-》我的服务-》修改OAuth 2.0网页授权为 你的域名 www.abc.com（不带http://）。

中移智能微信通 －－ 会员绑定功能使用方法
-----------------------------------------------------

绑定会员对于微信的粉丝有什么用呢？
1.直接在微信中查询订单，了解订单最新状态
3.微信搜索产品、或者使用某个功能增加积分
4.积分抽奖
绑定原有中移智能网站的会员，绑定之后可以继续使用原有会员的积分、等级、余额。提示：不绑定会员也可以使用免登录功能，绑定之后将会以绑定的会员名称登录
2.直接查询订单的快递情况，不需要输入快递单号


会员绑定分为：流程绑定、web绑定

1 流程绑定
输入:bd  入进绑定流程
输入用户名
输入密码
绑定流程中退出可以输入：ko

2 web绑定 是指进入网站登录绑定
